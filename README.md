# amelliya.com

## 📟 Technology Stack

- **[AstroJS](https://astro.build/)**
- **[MDX](https://mdxjs.com/)**
- **[Pagefind](https://pagefind.app/)**
- **[Expressive Code](https://expressive-code.com/)**

## 🎨 Features

- Fully responsive, mobile-first design
- Progressively enhanced
- Accessibility focused and i18n ready
- Privacy respecting
- Headless content with a nested Git repository

## 👩🏽‍💻 Developer

For future me, 

```bash
git clone https://github.com/amelliyaa/amelliya.com.git
cd amelliya.com/
npm install
```

> [!IMPORTANT]
> Don't forget to clone the content repository!

| Command                   | Action                                      |
| :------------------------ |:--------------------------------------------|
| `npm run dev`             | Starts local dev server at `localhost:4321` |
| `npm run build`           | Builds production site to `./dist/`         |
| `npm run preview`         | Previews built site locally                 |
