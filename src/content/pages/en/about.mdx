## This website

Was designed and built by me as a place to document my journey in the world of game development. Third party notices for any open source components used are available in the <Link href="/credits.txt">credits.txt file</Link>.

### Technologies

<Link href="https://astro.build/" external={true}>Astro</Link> is my static site generator of choice. The <abbr title="HyperText Markup Language">HTML</abbr> is housed within <code>.astro</code> component files and styled with scoped <abbr title="Cascading Style Sheets">CSS</abbr>. A minimal amount of JavaScript is used to provide <Link href="https://developer.mozilla.org/en-US/docs/Glossary/Progressive_Enhancement" external={true}>progressive enhancement</Link>. The pretty syntax highlighting on code blocks is thanks to the <Link href="https://github.com/shikijs/shiki" external={true}>Shiki</Link> library (using the Dracula theme). Blog posts are written in <Link href="https://github.com/mdx-js/mdx" external={true}>MDX</Link>.
